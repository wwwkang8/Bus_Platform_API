# Bus_Platform_API
도심형 모빌리티 버스 플랫폼 API 입니다

# Bus API
기존의 카풀, 택시 어플리케이션은 많지만 일정 지역을 오고 가는 맞춤형 버스 어플리케이션이 없다는 것에 착안하여 맞춤형 셔틀버스 API를 개발한다.

# 1차 요구사항
- 회원가입
- 로그인/로그아웃
- 버스 예약 시스템
- 배차 목록 보여주기 기능

# 사용기술
- Spring Boot
- JPA
- Mysql
- AWS EC2, Travis CI
